# Password Strength Checker

This is a simple Python project that checks the strength of a password.

The program analyzes a password based on:

* Length of the password
* Presence of uppercase letters
* Presence of lowercase letters
* Numbers
* Special characters

Based on these checks, the program calculates a **password strength score** and classifies it as:

* Weak Password
* Medium Password
* Strong Password

## Features

* Password strength scoring
* Security suggestions for stronger passwords
* Built using Python
* Uses regular expressions (re module)

## How to Run

1. Clone this repository
2. Run the Python file

```bash
python password_checker.py
```

## Example

Input:

```
Enter your password: Khushi@123
```

Output:

```
Password Score: 5/5
Strong Password
```

## Technologies Used

* Python
